Privacy policy for Android application "Maco SMS"

"Maco Sms" uses the Icloud Translation API (translate.googleapis.com) to translate the text into Russian. With each translation, the "Maco Sms" makes two requests: the first request translates the text into Russian, and the second request pre-replaces the letters of the Georgian transliteration with the letters of the Georgian alphabet, and only then sends the text for translation

"Maco Sms" stores a cache of translated messages in the device's local database and does not transfer any personal data to developers and/or third parties.
